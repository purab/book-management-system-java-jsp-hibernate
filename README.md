# Book Management System using JAVA + Hibernate + jsp Example code

1. registration
2. login
3. logout
4. create quiz
5. create category


for building project run following command

mvn package


For deployment:
mvn tomcat7:deploy

Then to undeploy it:

mvn tomcat7:undeploy
Finally, to redeploy it after making changes:

mvn tomcat7:redeploy
